替换

```
ls -al `which ls` 或
ls -al $(which ls)
```

高亮显示 grep

```
locate ls.c | grep ls.c
```

查找 find

```
find ~ -name "*.c"
```

.c .cpp .sh .h 求总行数

```
sudo find ~ \( -name "*.[ch]" -o -name "*.sh" -o -name "*.cpp" \) -exec wc -l {} \; | xargs wc -l
```
